#List-stack Object

The `tables` module creates common table patterns.

	<ul class="list-stack">
		<li></li>
		<li></li>
		<li></li>
	</ul>

## Installation

Install using Bower:

	$ bower --save bare-table

Import into project:

	@import "bower_components/bare-table/objects.table";


## Options

**table--cells**

The `table--cells` option adds a border around table cells.

	<table class="table table--cells">
		<thead>
			<tr>
				<th></th>
				<th></th>
				<th></th>
			</tr>
		</thead>	
		<tbody>
			<tr>
				<td></td>
				<td></td>
				<td></td>
			</tr>
		</tbody>
	</table>

**table--rows**

The `table--rows` option adds a border above table rows.

	<table class="table table--rows">
		<thead>
			<tr>
				<th></th>
				<th></th>
				<th></th>
			</tr>
		</thead>	
		<tbody>
			<tr>
				<td></td>
				<td></td>
				<td></td>
			</tr>
		</tbody>
	</table>

**table--fixed**

The `table--fixed` option makes columns equal in width.

	<table class="table table--fixed">
		<thead>
			<tr>
				<th></th>
				<th></th>
				<th></th>
			</tr>
		</thead>	
		<tbody>
			<tr>
				<td></td>
				<td></td>
				<td></td>
			</tr>
		</tbody>
	</table>

**table--zebra**
	
The `table--zebra` option applies zebra striping to table rows.

	<table class="table table--zebra">
		<thead>
			<tr>
				<th></th>
				<th></th>
				<th></th>
			</tr>
		</thead>	
		<tbody>
			<tr>
				<td></td>
				<td></td>
				<td></td>
			</tr>
		</tbody>
	</table>
